# Trim lines in `JSON` string

This function takes a `JSON` string as input and returns a modified
version of the input where the values portion of each line is trimmed
for a less messy console output.

## Usage

``` r
trim_lines_json(x)
```

## Arguments

- x:

  (`character`)

## Value

A character string trimmed after a certain hard-coded number of
characters in the value portion.
